# Biblion – Biblioteca Digital

**Biblion** es una aplicación web para la gestión de una biblioteca digital. Permite a los usuarios registrarse, iniciar sesión, consultar el catálogo de libros, agregar nuevos títulos, editar y eliminar libros, así como llevar un historial de lecturas. Está diseñada como un frontend que consume una API REST desplegada en la nube.

---

## Tecnologías utilizadas

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
- **Backend**: API REST desarrollada con Node.js + Express (desplegada en Render)
- **Base de datos**: MySQL (administrada por el backend)
- **Comunicación**: Fetch API para consumir los endpoints REST
- **Control de versiones**: Git + GitHub
